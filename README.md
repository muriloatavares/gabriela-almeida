## 🚀 Como Usar

### Instalação Local

1. **Clone ou Download do Projeto**
   ```bash
   cd "Gabriela Almeida"
   ```

2. **Abra no VS Code**
   ```bash
   code .
   ```

3. **Inicie um Live Server**
   - VS Code: Extensão "Live Server" (Right Click → Open with Live Server)
   - Terminal: `python -m http.server 8000`
   - Acesse: `http://localhost:8000`

### Customização

#### Mudar Cores da Marca
No `index.html`, linha ~30:
```javascript
colors: {
    brand: {
        dark: '#2E2433',      // Roxo escuro
        text: '#1E1A1D',      // Preto suave
        nude: '#EBD8D0',      // Rosa claro
        lilac: '#B8AFC4',     // Lilás
        green: '#C3CEC3',     // Verde
        light: '#FAFAFA'      // Fundo
    }
}
```

- Modifique o HTML na seção correspondente

## 🐛 Troubleshooting

### Imagens não aparecem
- Verifique caminho: `assets/images/Gabriela Almeida/`
- Confirme espaços e acentuação no caminho
- Limpe cache do navegador (Ctrl+Shift+Delete)

### Parallax travado no mobile
- Velocidade reduzida a 0.15-0.45 (otimizado)
- Se lento: desabilitar parallax em mobile < 768px

### Animações stuttery
- Verificar CPU/RAM disponível
- Desabilitar outras extensões
- Usar navegador moderno (Chrome 90+)
